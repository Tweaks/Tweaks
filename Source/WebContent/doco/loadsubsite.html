Load Subsite

Description
This tweak will replace the content area of a page with a fully functional view of another Blackboard Site's content area. It can be useful to load sections of a shared site, without appearing to leave the original site.
Skill Level
Easy (No HTML)
How to Use

  1. Copy the direct link to a folder or content area of another Blackboard site to your clipboard.

    You will need to make sure that you have Edit Mode: OFF, otherwise you will be copying a link that Students / Participants won't be able to access.

    Try visit the link directly by pasting it in your Web Browser to test. You should be taken to the page that already has Edit Mode: OFF and is missing a Blackboard banner across the top of the screen.

  2. Navigate to the site and content area / folder that you wish to show this Subsite.

  3. Create an External Link titled 'Site Link', and paste in the URL to the sub site section

  4. Switch Edit Mode: OFF to test the tweak.
