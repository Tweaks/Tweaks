Row Hider

Description

This tweak will make a row invisible with Edit Mode: OFF.

This can be used to hide items that you do not want visible to students but need to be available to retain functionality (eg: items containing CSS, Scripts or other content used on the page).

Skill Level
Medium (Simple HTML)
How to Use

  1. Edit any item.

  2. In the text editor, toggle HTML Source Mode add the following code anywhere:

    <div class="hidemyrow">row</div>
