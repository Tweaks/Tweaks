Submenu

Description
This tweak adds a horizontal submenu based on the content items and subfolders of the current page. The menu can be text only, text overlaying an image or inserted into custom HTML.
Skill Level
Advanced (Requires HTML)
How to Use
  1. Add your content items and folders to your page.
  2. Add the tweak at the base of the page
  3. Optional: Add a Content item called 'Menu Image' and insert an image in the content area of the item. Make sure to include image height, width, alt text (and optionally CSS class). The image should be wide enough to accommodate the text of the menu items
