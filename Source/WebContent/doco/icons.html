Add Coloured Icon Box

Description
This tweak adds a coloured box with an icon around a block of text.
Skill Level
Medium (Simple HTML)
How to Use

  1. Edit any item, then in the text editor toggle HTML Source Mode

  2. Wrap the text you wish to highlight inside a HTML <div> element, eg:

    <div class='becareful'>Remember to click the 'attach document' button.</div>

  3. Optional: Replace 'becareful' with another valid box name

Additional Options

The following are valid box names:

information magnify message tickblue back next tickgreen becareful issue help
