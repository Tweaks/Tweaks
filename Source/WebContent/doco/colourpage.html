Colour Page

Description
This tweak changes the background colour of the page
Skill Level
Medium (Simple HTML)
How to Use

  1. Edit any item, then in the text editor toggle HTML Source Mode

  2. Add the following HTML anywhere:

    <span id="colourPage">colourcode</span>

  3. Replace "colourcode" in the above example with a valid HTML hex colour you want for the background of the page.

Note: There are a number of online tools available to help choose colour codes, such as colourpicker.com
