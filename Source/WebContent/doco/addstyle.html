Add CSS Stylesheet

Description
This tweak allows you to load a CSS Stylesheet to the page where this tweak is added.
Skill Level
Advanced (Simple HTML, Custom CSS)
How to Use
  1. Create a new Blackboard item named "Stylesheet".
  2. Use the Browse My Computer button to upload your custom .css stylesheet file, or click Browse Unit to reuse a stylesheet you have already added to the site.
  3. Save the item. This row will then be hidden with Edit Mode: OFF.
Additional Options

Optional: Instead of creating a new "Stylesheet" item, you can add the following HTML markup to any existing content item:

<div class='loadStyle'>Relative URL to your .css File</div>
