Replace or Hide Icons

Description
This tweak allows you to replace or hide the standard Blackboard icons.
Skill Level
Easy
How to Use

  1. Add a Blackboard item titled one of the following:

    Standard Icon Replacements

    • Everything Replacement Icon
    • Item Replacement Icon
    • Folder Replacement Icon
    • Title Replacement Icon

    Custom Icon Replacements

    • Replacement Icon: Row Name

    Hiding Icons

    • Hide Icon: Row Name

    Note: Replace Row Name with the full or partial title of item/s you wish to change the icon for.

  2. Use the Add Image button in the Text Editor to upload your custom icon, unless you are using "Hide Icon".
Advanced Usage (Simple HTML)

This tweak also accepts using simple HTML to change icons.

  1. Insert the image using the Text Editor to the item itself, then using the HTML mode add <img class="replacemyicon" src="..." alt="..."> to the IMG element you just added

  2. Add <div class="hidemyicon">Hidden Icon</div> anywhere in HTML mode to a content item to hide it's icon
