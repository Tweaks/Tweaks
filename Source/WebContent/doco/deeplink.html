Make Deep Link

Description

This tweak will automatically fix direct links to Blackboard folders, ensuring they open correctly in a new window.

Note: Use Blackboard's native "Unit Links" to link to items within the same unit, as direct links may break when performing a Unit Copy.

Skill Level
Easy (No HTML)
How to Use

  1. Add a Blackboard item named 'Unit Map'. Use the Text Editor to insert an Image, and then Save this item.

  2. Move this new item with your image above the items you wish to have buttons automatically

  3. Create an External Link titled 'Site Link', and paste in the URL to the sub site section

  4. Switch Edit Mode: OFF to test the tweak.
