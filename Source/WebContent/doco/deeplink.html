Make Deep Link

Description

This tweak will automatically fix direct links to Blackboard folders, ensuring they open correctly in a new window.

Note: Use Blackboard's native 'Unit Links' to link to items within the same unit, as direct links may break when performing a Unit Copy.

Skill Level
Easy (Some HTML)
How to Use
  1. Find a link to another site's content area by right-clicking it's link in the left hand navigation, or on the title of a folder in the site
  2. Create a 'Web Link' on the page you wish
  3. In the text editor toggle HTML Source Mode and add the following HTML: 
    <span class="makeDeepLink">
  4. Click Submit, then add this tweak to the page to activate.
