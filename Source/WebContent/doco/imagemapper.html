Image Mapper

Description
This tweak dynamically maps Blackboard resources (folders and links) into an Image Map.
Skill Level
Advanced (Moderate HTML, Image Creation)
How to Use
  1. Create an image or find a copyright-free image to use
  2. Create your folders, links or other content items in Blackboard that your image map will link to.

  3. Generate an image map for your image. Use a web editor such as Dreamweaver (Instructions) or an online image map editor such as maschek.hu/imagemap.

    Make sure you add the exact titles of the folders or links to the ALT tag for each image map area. You can leave href="#"

  4. Create a Blackboard item and add the image using the 'Attach Image' button.

  5. Toggle HTML Source Mode, and add your image map HTML. Make sure to add the usemap="#MyImageMap" attribute to the image.

    Your image map should look similar to this:

    <img border="0" src="[BLACKBOARD IMAGE URL]" alt="My Image Map" usemap="#MyImageMap"/>
    <map name="MyImageMap">
    <area shape="rect" coords="31,46,100,77" href="#" alt="Link 1"/>
    <area shape="rect" coords="31,106,100,137" href="#" alt="Link 2"/>
    <area shape="rect" coords="398,48,467,79" href="#" alt="Link 3"/>
    <area shape="rect" coords="398,108,467,139" href="#" alt="Link 4"/>
    </map>
