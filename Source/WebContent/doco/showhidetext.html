Show Hide Text

Description
This tweak replaces sections of text with a 'Show/hide text' link
Skill Level
Medium (Simple HTML)
How to Use
  1. Create a new item in Blackboard, or edit an existing one.
  2. Write in the Text Editor all the content for this item, then toggle HTML Mode.

  3. Put the following HTML around any text that you wish to replace with a "Show/hide text" link:

    <div class='hidden'>Your content here</div>
  4. Save and test your tweak.

Optional:

To replace the "Show/hide text" link with a question or other text, use the following HTML immeadiatley before the hidden text: 

<span id='showHideInstruction'>Your own hide show instruction here</span>
