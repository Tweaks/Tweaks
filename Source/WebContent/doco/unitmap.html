Dynamic Unit Map Table

Description
This tweak dynamically maps Blackboard Content Items into a table, based on specially named titles
Skill Level
Medium (Optional HTML, Content Organisation)
How to Use
  1. Create or rename existing items, folder and links on your page with titles that follow this format:

    2. row: column name of resource

    Make sure that the : (colon) character is followed by a space

     

    Examples:
    • Week 1: Lecture Introduction to Schematics
    • Week 2: Lecture Schematics Process
    • Resources: Readings Future Workforce

  2. Create a new Blackboard item, and name it Unit Map.

  3. In the text editor for this new Unit Map, create a table with the right number of rows and columns for your content.

  4. Once you have added the table, fill in the row and column names that you used on your Blackboard Items, leaving the first upper-left cell blank.

    You do not need to change the format of the table, or type into the other cells.

  5. Save and check your table. The tweak will then move items that are correctly named into the table.

Additional Options

This tweak also has additional options for changing the table colour, highlighting cells, collapsing rows together into modules and more.

These extra features do require using the HTML editor, and instructions can be found through the detailed examples link below for more information.
