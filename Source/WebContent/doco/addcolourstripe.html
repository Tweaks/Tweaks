Add Colour Stripe

Description
This tweak adds a colour stripe to a Blackboard item.
Skill Level
Medium (Simple HTML)
How to Use

  1. Edit any item.

  2. In the text editor, toggle HTML Source Mode add the following code anywhere:

    <span class='colourSpan'>#99BBCC</span>

  3. Replace #99BBCC with a valid HTML hex colour.

    You can use www.colorpicker.com to help you choose a colour code.
