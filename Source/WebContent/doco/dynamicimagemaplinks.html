Dynamic Unit Map Image

Description

This tweak dynamically overlays any image with button-style links to content items (folders, links, tools etc).

It positions and styles the links based on predesigned stylesheet options.

Skill Level
Medium (Image Creation, Content Organisation)
How to Use
  1. Create an image or find a copyright-free image to use. Your image should be 770px (pixels) wide.
  2. Create a Blackboard item named Unit Map for a vertical structure or Unit Navigation for a horizontal structure
  3. Add your image to this new item and then save.
  4. Create your folders, links or other content items that you wish to add as buttons to this image. Their order on the page will determine the appearance of the Visual Unit Map.
Additional Options

The following extra steps can be used to customise the layout and size of your buttons:

Increase the font size of the link

  • Use "Module..." for a folder's name (this will also add a column break)
  • Use any folder/link name and add "column header" to its description

Add a Column Break / Start a new Column

  • Use "Module..." for a folder's name (this will also increase it's size)
  • Use any folder/link name and add "column break header" to its description

Add a Row Break / Start a new Row

  • Use any folder/link name and add "row break" to its description
