Checklist

Description
This tweak will move the Review buttons on items using Blackboard's native 'Set Review Status (Enabled)' setting into a simple checklist.
Skill Level
Easy
How to Use
  1. Add an item called 'Checklist'.
  2. Enable the Review Status setting by clicking on the context menu icon next to another item and selecting Set Review Status, then on the next page select "Enabled".
