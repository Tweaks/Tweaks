Tweaks by Tweaks

Tool to customisate and extend Blackboard LMS.

Tweaks is a Building Block for the Blackboard LMS which enables academics and support staff to customise and extend their site presentation, navigation and structure as well as add templated page designs, small learning activities and Blackboard functionality.

Dependencies

jQuery (included)

Install

Download and install Building Block via Blackboard System Admin. Alternatively, if you are a site coordinator and you want to trial a Tweak, you can copy and paste HTML (in development) into sites.
