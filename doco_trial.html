Trialing a Tweak on your site:

  1. Select the Tweak to see documentation and a screenshot, then click "Toggle: test code to embed in Blackboard site."
  2. Copy the text from the yellow box that appears.
  3. In your Blackboard site, create a test folder not available to students:



  4. Create a new Blackboard item inside this folder and call it "Remote Tweak". Switch the visual editor OFF (as per image below) and paste in the text you copied.



  5. Your Tweak should now be ready to use as per the instructions on the main Documentation page.
