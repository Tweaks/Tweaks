Tweaks: Documentation: Trialling Tweaks

Trialling Tweaks on your Blackboard site:

  1. From the Documentation page, select a Tweak to see usage instructions and a screenshot, then click "Toggle: Trial Code" at the base of hte page.
  2. Copy the text from the yellow box that appears.
  3. In your Blackboard site, create a test folder that is unavailable to students:



  4. Create a new Blackboard item inside this folder and call it "Remote Tweak". Switch the visual editor OFF (as shown below) and paste in the text you copied.



  5. Your Tweak should now be ready to use as per the instructions on the main Documentation page.
